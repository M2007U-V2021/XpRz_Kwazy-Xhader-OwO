Fragment Shader Concept

This shader uses both the ForwardBase Pass and ForwardAdd Pass

what is a Pass ?

Imagine we have the following setup : an empty canvas and 3 robots Each robot is designed to paint differently on the same canvas based on some different algotrithyms and trigger conditions

Each Robot in this example is a Pass Each Pass is designed to do something specifically. It acts like the layers in Photoshop

what is ForwardBass ?

ForwardBass is a Pass in Unity, which is the first Pass that Unity will run, and it holds the base color of the material. ForwardBass can also control how a single realtime directional light can affect a mesh with the material

what is ForwardAdd ?

ForwardAdd is a another Pass in Unity, which will be trigger/called for every realtime light that is in range to shine the mesh with the material

Layers in KXOwO

in KXOwO, we have 5 layers, that each has its own purpose

Layer NamePassDesc
Texture Layer 1ForwardBaseProvide texture colors, the main albedo of the shader, can be shader animated
Texture Layer 2ForwardBaseThe same as Texture Layer 1. Both Texture Layer 1 and Texture Layer 2 will be blended together with different blend types
Glitter LayerForwardBaseAdds a glitter effect on top of the stack, can be shader animated
Lightmap LayerForwardBaseAdds a layer of Unity Baked Lightmap, texture needs to be added manually
Runtime LightForwardAddAdding runtime shades and color when illuminated by realtime point lights
caution

the Lightmap Layer will only work for 2D meshes

To controll how intense each layer is, the Kontrol Panel has the sliders for each Layers

POWAH!!!

when a layer is to intense, it might cause Color Flipping, make sure the strength adds up to 1 (idealy) or below 2, Unless you want that intensive color flip effect

Layer Bwend Mode

this is the option to choose how to combine both Texture Layer 1 an Texture Layer 2

ModeDesc
Additiveadding both layers together Channel by Channel
First on TopJust like photoshop layers, Texture Layer 1 is on top of Texture Layer 2
Second on TopSimiliar to First on Top, but it is Texture Layer 2 on Top instead

Blend Mode

This is where you can adjust how Unity will blend this KXOwO Passes with other External Shaders

  • Bwend Mode Sos : the "Source" in blending
  • Bwend Mode Beehind : the "Destination" in blending
  • Bwend Mode Opewation : the "Operation" in blending
Example

most of the time we usually use the blending method : SrcAlpha OneMinusAlpha with the operation Add

  • Kull Taip : basically "Cull Type"
Cull Type

If you have ever tried to import an FBX from Blender into Unity, You might have came to the situation that some faces are missing, most of the time, the faces are not missing, they are just facing the wrong way.

In this case, Cull Type take cares for that.

  • Off : Render Both sides
  • Front : Render Front faces only
  • Back : Render Back faces only
