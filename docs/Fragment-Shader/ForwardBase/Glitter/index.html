Glitter Layer

in KXOwO, you can add a layer of Glitters to make your meshes go "blink blink OwO <3"

And here's how it looks like in Unity :

caution

if you are looking for a perfect glitter effect, what you need here is a noise texture.
In this case, to generate a good noise texture for this situation, the algorythym is to have the channels be a random number between 0-255 for each pixel , here's an example of a noise texture that is a suitable texture to be used for the Glitter layer

Noise Teksture Hue Scwoll / Noise Texture Hue Scroll​

Basically the rate for the "Blinking" effect, the higher the number, the faster it sparkles

Gwitter Thweshhold / Glitter Threshhold​

First we need to know how a glitter to pixel is made :

/*
note that
GwitterThresholdRwed,
GwitterThresholdGwin,
GwitterThresholdBwoo,
GwitterThresholdAfwa 
are all user adjustable parameters
*/


bool boolRed;
bool boolGreen;
bool boolBlue;
bool boolAlpha;

boolReed = HueShift(ColorFromNoiseTexture.r) > GwitterThresholdRwed;
boolGwin = HueShift(ColorFromNoiseTexture.g) > GwitterThresholdGwin;
boolBwoo = HueShift(ColorFromNoiseTexture.b) > GwitterThresholdBwoo;
boolAfwa = HueShift(ColorFromNoiseTexture.a) > GwitterThresholdAfwa;

if(boolRwed && boolGwin && boolBwoo && boolAfwa)
{
    //this pixel passes the test
    //this pixel will be colored with the Glitter color
}
else
{
    //this pixel did NOT pass the test,
    //this pixel will not be colored with the Glitter Color
}
Why it is not Glittering ?

If all the threshhold sliders are the same, then there is not going to be any glittering animation, in order to let it work, all threshold sliders must have different values
