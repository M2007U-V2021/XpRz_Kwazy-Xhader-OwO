External - Fresnel

Basically saying, a Fresnel effect can make a mesh face facing away from or at you to be colored with a color

Example 1

We can use this to make fake glass, works extremely good with spheres, or convex meshes

Example 2

We can also use Fresnel to shade meshes

note

In KXOwO, there are 2 Fresnel Layers, and each of them can be masked (i.e. they have their own masking layer)

to activate it

To make a Fresnel Layer Effect visible, you will need to let atleast one Texture Layer be influenced by the Fresnel Effect as Fresnel is considered to be an influencer on the Texture Layer. In this case, goto Main Texture Layer, there you can find the slider Fwensel Infwuence, crank that bad boi up

here's a snapshot of a Fresnel effect on a sphere:

Force Alpha

under Main Texture Layer, where you adjust how much the Layer is influenced by Fresnel, there is another slider Fwensel ForceAfwa, this will make the fresnel result be transparent or opaque

Wejen Invert / Region Invert​

By changing this parameter, we can change the region to be colored.
Here's a series of snapshots with the region invert parameter being 0.2, 0.4, 0.5, 0.6, 0.8, 1.0

tip

by default the calculation for this would be
dot(MeshNormalVector,CameraViewVector) = strength
by flipping the region, it is just
1 - dot(MeshNormalVector,CameraViewVector) = strength

PS: here, both the MeshNormalVector and the CameraViewVector are normalized

Ramp​

this controll how rough/glossy the fresnel is, or if you are using this on a sphere, it can be interpreted as "how thick is the ring going to be"

Here's a series of snapshots with the Ramp parameter being 0, 1, 2, 3, 4, 8

Tweshhold Low / Hgh​

We can use this to cut off unwanted parts of the fresnel, This uses the Threshhold Color Transform Operation, with ValueLow locked to 0 and ValueHgh locked to 1

Here's a series of snapshots with the Threshold Low parameter being 0.0, 0.1, 0.2, 0.3

Here's a series of snapshots with the Threshold High parameter being 0.005, 0.1, 0.2
